# 下载说明

不要点仓库右上角 `Code -> Download ZIP` 来下载整个源码仓库。

普通用户直接点击下面对应系统的安装包 `.zip` 即可：

## OpenClaw

- `01-OpenClaw-Windows一键安装包.zip`
- `02-OpenClaw-macOS一键安装包.zip`
- `03-OpenClaw-Linux一键安装包.zip`

## Codex + Claude

- `04-CodexClaude-Windows一键安装包.zip`
- `05-CodexClaude-macOS一键安装包.zip`
- `06-CodexClaude-Linux一键安装包.zip`

下载后：

1. 解压 zip
2. 打开解压后的文件夹
3. 运行里面的启动文件

Windows 用户运行：

- `Windows-双击我-安装OpenClaw配置.cmd`
- `Windows-双击我-安装Codex和Claude配置.cmd`

macOS 用户运行：

- `macOS-双击我-安装OpenClaw配置.command`
- `macOS-双击我-安装Codex和Claude配置.command`

Linux 用户运行：

- `Linux-终端版-安装OpenClaw配置.sh`
- `Linux-终端版-安装Codex和Claude配置.sh`
